# <a name="rundll32-printuidllprintuientry"></a>rundll32.exe printui.dll，PrintUIEntry

>适用于：Windows Server （半年频道），Windows Server 2016，Windows Server 2012 R2，Windows Server 2012

自动执行许多打印机配置任务。 printui.dll 是包含打印机配置对话框使用的函数的可执行文件。 这些函数也可以从脚本或命令行批处理文件中调用，也可以从命令提示符以交互方式运行。 有关如何使用此命令的示例，请参阅[示例](#BKMK_Examples)。  
## <a name="syntax"></a>语法  
```  
rundll32 printui.dll PrintUIEntry [BaseParameter] [ModificationParameter1] [ModificationParameter2] [ModificationParameterN]  
```  
你还可以使用以下替代语法，虽然本主题中的示例使用上面的语法：  
```  
rundll32 printui.dll,PrintUIEntry [BaseParameter] [ModificationParameter1] [ModificationParameter2] [ModificationParameterN]  
```  
```  
rundll32 printui PrintUIEntry [BaseParameter] [ModificationParameter1] [ModificationParameter2] [ModificationParameterN]  
```  
```  
rundll32 printui,PrintUIEntry [BaseParameter] [ModificationParameter1] [ModificationParameter2] [ModificationParameterN]  
```

## <a name="remarks"></a>备注  
- **PrintUIEntry**关键字区分大小写，并且必须以本主题中的示例中所示的大小写形式输入此命令的语法。  
- 有关某些常见任务的语法，请参阅本文档中的[示例](#BKMK_Examples)。 有关更多示例，请在命令提示符下键入： **rundll32.exe printui.dll，PrintUIEntry/？**  
  ## <a name="BKMK_Examples"></a>示例  
  若要为计算机 Client1 （对运行此命令的用户帐户可见）添加新的远程打印机 printer1，请键入：  
  ```  
  rundll32 printui.dll PrintUIEntry /in /n\\client1\printer1  
  ```  
  若要使用 "添加打印机向导" 来添加打印机，并使用位于驱动器 c： at Infpath 的 .inf 文件，请键入：  
  ```  
  rundll32 printui.dll PrintUIEntry /ii /f c:\Infpath\InfFile.inf  
  ```  
  若要删除现有打印机 printer1，请在计算机 Client1 上键入：  
  ```  
  rundll32 printui.dll PrintUIEntry /dn /n\\client1\printer1  
  ```  
  若要为计算机的所有用户添加每台计算机的打印机连接 printer2，请键入（用户登录时将应用该连接）：  
  ```  
  rundll32 printui.dll PrintUIEntry /ga /n\\client2\printer2  
  ```  
  若要为计算机的所有用户删除每台计算机的打印机连接 printer2，请键入（用户登录时将删除该连接）：  
  ```  
  rundll32 printui.dll PrintUIEntry /gd /n\\client2\printer2  
  ```  
  若要查看打印服务器的属性 printServer1，请键入：  
  ```  
  rundll32 printui.dll PrintUIEntry /s /t1 /c\\printserver1  
  ```  
  若要查看打印机的属性，请 printer3，键入：  
  ```  
  rundll32 printui.dll PrintUIEntry /p /n\\printer3  
  ```
